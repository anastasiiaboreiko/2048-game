# 2048 Game

**2048 Game** is a browser-based puzzle game inspired by the classic 2048 concept. The game challenges users to combine numbered tiles on a grid to reach higher values, with the ultimate goal of creating the 2048 tile.

The main purpose of the project is to provide an engaging and interactive user experience while demonstrating core game mechanics, logical thinking, and smooth user interaction.
